## 关于行星科学知识库

**行星科学知识库**（Planetary Science Knowledge Hub）是一个致力于行星科学教育与资源共享的开放平台。

### 目标

- 为行星科学学习者提供**系统化、结构化的知识整理**
- 汇集**领域内的优质资源与学者信息**
- 促进**行星科学知识的传播与共享**

### 技术栈

本站使用以下技术构建：

- [VuePress 2](https://vuepress.vuejs.org/) — 静态网站生成器
- [vuepress-theme-hope](https://theme-hope.vuejs.press/) — 功能强大的 VuePress 主题
- [GitHub Pages](https://pages.github.com/) — 静态网站托管

### 网站特色

- **系统化知识**: 基于经典教材与文献，构建行星科学知识体系
- **Markdown 友好**: 所有内容使用 Markdown 编写，支持数学公式、定理环境与图表
- **Obsidian 兼容**: 学科知识文档可在 Obsidian 中编辑和预览，与网站渲染效果一致
- **开源共享**: 代码与内容完全开源，欢迎贡献与反馈
- **GitHub Pages 部署**: 自动构建，持续更新

### 内容来源

本站学科知识板块的内容基于以下经典教材与文献整理：

- 《数学物理方法》相关教材
- 《回归分析》统计方法教材
- 《宇宙化学》相关专著
- 《行星科学》导论教材
- 地球物理学与遥感技术相关文献

### 如何贡献

如果你有建议或想要贡献内容，欢迎通过以下方式参与：

- 在 [GitHub](https://github.com/your-username/planetary-science) 上提交 Issue 或 Pull Request
- 使用 Obsidian 编辑 `docs/knowledge/` 下的 Markdown 文件，内容将自动同步到网站
- 联系管理员推荐新的学习资源或学者信息
