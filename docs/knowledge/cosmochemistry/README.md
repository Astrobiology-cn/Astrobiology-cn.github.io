# 宇宙化学

> 宇宙化学是研究宇宙中天体化学组成和演化过程的学科

## 概述

宇宙化学（Cosmochemistry）研究太阳系及宇宙中元素的丰度、分布和演化规律。它是行星科学的重要基础学科之一。

## 主要章节

- **元素起源** — 恒星核合成、大爆炸核合成、超新星核合成
- **太阳系化学演化** — 太阳星云、原始太阳、行星分馏过程
- **陨石分析** — 陨石分类、同位素年代学、前太阳颗粒
- **行星大气化学** — 大气逃逸、光化学、大气演化模型
- **表面化学过程** — 风化、辐射作用、空间风化

## 应用场景

在行星科学中，宇宙化学广泛应用于：

- 太阳系形成年代与演化历史重建
- 行星宜居性评估与生命起源研究
- 陨石与彗星物质分析
- 行星内部结构与成分反演
