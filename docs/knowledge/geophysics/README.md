# 地球物理学

> 地球物理学方法在行星探测中的应用

## 概述

地球物理学利用物理方法研究行星内部结构、重力场、磁场、地震波等物理性质，是深空探测和行星科学研究的重要工具。

## 主要章节

- **重力场与形状** — 重力位、球谐展开、大地水准面
- **磁场与磁层** — 发电机理论、磁异常、太阳风相互作用
- **地震学** — 地震波传播、行星内部结构反演
- **电磁学** — 电磁感应、内部电导率
- **热流与热演化** — 热传导、对流、热历史建模

## 应用场景

- 月球、火星等天体的内部结构探测
- 行星重力场与形状建模
- 行星磁场与空间环境研究
- 行星宜居性评估与资源勘探
