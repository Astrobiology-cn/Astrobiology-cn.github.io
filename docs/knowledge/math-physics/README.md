# 数学物理方法

> 行星科学中数学物理方法的核心内容整理

## 概述

数学物理方法是行星科学研究的重要工具，涵盖了偏微分方程、特殊函数、复变函数等数学工具在物理问题中的应用。

## 主要章节

- **复变函数基础** — 解析函数、留数定理、积分变换
- **偏微分方程** — 分离变量法、格林函数、特征值问题
- **特殊函数** — 勒让德多项式、贝塞尔函数、球谐函数
- **积分变换** — 傅里叶变换、拉普拉斯变换及其应用
- **变分法** — 泛函极值、欧拉-拉格朗日方程

## 应用场景

在行星科学中，数学物理方法广泛应用于：

- 行星重力场与磁场建模
- 热传导与热演化计算
- 行星地震波传播分析
- 流体动力学与大气环流模拟
