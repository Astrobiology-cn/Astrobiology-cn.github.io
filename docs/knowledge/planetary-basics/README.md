# 行星科学基础

> 行星科学的基础理论与核心概念

## 概述

行星科学（Planetary Science）是一门跨学科科学，研究行星、卫星、小行星、彗星等太阳系天体及其形成、演化与物理过程。

## 主要章节

- **行星形成** — 吸积理论、原行星盘、迁移
- **行星内部结构** — 分层、状态方程、热演化
- **行星大气** — 大气逃逸、温室效应、气候演化
- **行星表面** — 撞击、火山、构造、侵蚀
- **行星磁层** — 磁场产生、太阳风相互作用

## 应用场景

- 太阳系形成与演化历史重建
- 系外行星探测与宜居性评估
- 深空探测任务规划与数据解译
- 行星资源勘探与利用
