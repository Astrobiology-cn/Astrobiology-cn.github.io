# 回归分析

> 统计学与机器学习方法在行星数据中的应用

## 概述

回归分析是行星数据科学的核心工具，用于建立变量之间的关系模型，从观测数据中提取物理规律。

## 主要章节

- **线性回归** — 最小二乘法、多元线性回归、正则化
- **非线性回归** — 参数估计、迭代方法、拟合优度
- **时间序列分析** — 自回归模型、谱分析、趋势检测
- **机器学习基础** — 决策树、随机森林、支持向量机
- **深度学习应用** — 神经网络、CNN、RNN 在行星数据中的应用

## 应用场景

在行星科学中，回归分析广泛应用于：

- 行星参数反演（温度、成分、密度）
- 光谱数据解译与矿物识别
- 撞击坑统计与年代学
- 行星表面过程的时间演化建模
