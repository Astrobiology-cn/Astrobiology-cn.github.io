# 遥感技术

> 行星遥感原理与数据处理方法

## 概述

遥感技术（Remote Sensing）通过非接触方式获取行星表面的电磁波信息，是行星探测与地表研究的核心手段。

## 主要章节

- **电磁波基础** — 辐射传输、光谱学、反射与发射
- **光学遥感** — 可见光-红外成像、高光谱、多光谱
- **雷达遥感** — SAR、InSAR、穿透雷达
- **激光遥感** — 激光高度计、激光雷达
- **数据处理** — 几何校正、辐射校正、图像融合

## 应用场景

- 行星表面制图与地形测量
- 矿物识别与成分分析
- 大气成分探测与气候监测
- 撞击坑年代学与地质过程分析
