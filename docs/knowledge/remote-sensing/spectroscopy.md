# 光谱遥感基础

## 电磁辐射与行星表面相互作用

### 反射光谱

行星表面的反射光谱由以下因素决定：

- **矿物成分**: 不同矿物具有特征吸收带
- **颗粒大小**: 影响散射特性与光谱对比度
- **表面粗糙度**: 影响反射率与偏振特性
- **空间风化**: 长期暴露在空间环境中改变光谱特征

### 普朗克黑体辐射

热发射遵循普朗克定律：

$$
B_\lambda(T) = \frac{2hc^2}{\lambda^5} \frac{1}{e^{hc/\lambda k_B T} - 1}
$$

其中 $h$ 为普朗克常数，$c$ 为光速，$k_B$ 为玻尔兹曼常数，$T$ 为温度。

## 矿物光谱特征

### 常见矿物吸收带

| 矿物 | 主要吸收带 (μm) | 成因 |
|------|----------------|------|
| 橄榄石 | 1.05, 1.25 | Fe²⁺ 电子跃迁 |
| 辉石 | 1.0, 1.9, 2.3 | Fe²⁺ 电子跃迁 |
| 斜长石 | 1.25, 1.4 | O-H 振动 |
| 赤铁矿 | 0.86, 0.93 | Fe³⁺ 电子跃迁 |
| 水冰 | 1.5, 2.0, 3.2 | O-H 振动 |

### 光谱比值分析

通过比值运算增强光谱特征：

$$
R_{ratio} = \frac{R_{\lambda_1}}{R_{\lambda_2}}
$$

常用于识别矿物成分和校正光照效应。

## 遥感仪器类型

### 多光谱成像

- **Viking/MOC**: 火星轨道器相机，可见光成像
- **LROC**: 月球勘测轨道器相机，高分辨率成像
- **HiRISE**: 火星高分辨率成像科学实验，0.25 m/像素

### 高光谱成像

- **CRISM**: 火星紧凑型侦察成像光谱仪，覆盖 0.36-3.92 μm
- **OMEGA**: 火星快车可见光与红外矿物学光谱仪
- **Moon Mineralogy Mapper**: 月球矿物学制图仪

### 热红外遥感

- **THEMIS**: 火星热辐射成像系统，识别热惯性
- **DIVINER**: 月球勘测轨道器热红外实验，测量表面温度

## 应用实例：火星水冰探测

通过近红外光谱分析，在火星中纬度地区发现了水冰：

- **吸收特征**: 1.5 μm 和 2.0 μm 处的水冰吸收带
- **分布特征**:  Fresh 撞击坑暴露的地下冰
- **季节变化**: 二氧化碳霜的季节性覆盖与消融
